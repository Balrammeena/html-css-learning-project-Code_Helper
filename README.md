#  Code Helper

Code Helper is my **first HTML & CSS project**, created to practice web design fundamentals and build a simple learning platform for programming tutorials.

This website provides **direct links to tutorials and video resources** for popular programming languages and technologies.



##  Project Features

- Clean and simple UI
- Responsive layout (mobile, tablet, desktop)
- Navigation bar with multiple programming topics
- Search box (Google-based search)
- Tutorial section with video links
- Floating WhatsApp chat icon
- Professional footer with useful links


##  Technologies Used

- HTML5
- CSS3
- Basic Responsive Design (Media Queries)
